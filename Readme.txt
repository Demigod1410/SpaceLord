## 🎮 About

SpaceLord is my first game ever created using the Godot Engine! This space-themed adventure marks the beginning of my game development journey, and I'm excited to share it with the community. More games are planned for the near future as I continue to learn and grow as a developer.

## ✨ Features

- 🚀 Space-themed gameplay
- 🎨 Custom artwork and assets
- 🎵 Immersive sound effects
- 🎯 Engaging game mechanics
- 🌟 Built entirely in Godot Engine

## 🛠️ Built With

- **Engine**: [Godot Engine](https://godotengine.org/) (Version 4.x)
- **Language**: GDScript
- **Platform**: Cross-platform (Windows, macOS, Linux)

## 🚀 Getting Started

### Prerequisites

- Godot Engine 4.x or later
- Basic understanding of Godot project structure

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Demigod1410/SpaceLord.git
   cd SpaceLord
   ```

2. **Open in Godot**
   - Launch Godot Engine
   - Click "Import" and select the project folder
   - Open the `project.godot` file

3. **Run the game**
   - Press `F5` or click the play button
   - Select the main scene if prompted

## 📁 Project Structure
